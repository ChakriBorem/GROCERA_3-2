# 🛒 GROCERA – Smart Grocery Shopping Assistant

## 📌 Overview

**GROCERA** is an intelligent grocery inventory and shopping assistant that enables users to manage their groceries with ease and efficiency. With support for CSV uploads, expiry tracking, price prediction, and AI-powered insights, GROCERA helps reduce food wastage and optimize household stock management.


---

## 🚀 Key Features

- 🧾 Upload and process `.csv` grocery files (inventory, prices, expiry dates)
- ⏰ **Expiry Alerts** for perishable items
- 📈 **Price Prediction** using trained ML models
- 📊 Integrated **Exploratory Data Analysis (EDA)** notebooks
- 🧠 **Google Generative AI** integration for smart suggestions
- 🌐 **Web-based UI** built with Flask + HTML/CSS/JS templates
- 📦 Organized project structure with modular code and reusable components
